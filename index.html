Notice: The Magic happens mainly in the code of the SVG Files. The <symbol> tags have a viewbox directly implemented in the .svg file, so it's simpler to use anywhere on a html document with the <use> tag (see index.symbols.html). The group tag <g> doesn't have a viewBox attribute, therefore you need to define it everytime you use it in your html document (see index.groups.html), e.g. <svg viewBox=...>
